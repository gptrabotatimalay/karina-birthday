## 🚀 Запуск за 1 минуту

### Шаг 1: Установите зависимости (если ещё не установлены)

```bash
npm install
```

### Шаг 2: Запустите игру

```bash
npm run dev
```

### Шаг 3: Играйте!

Откроется браузер → http://localhost:3000

**Управление:**
- `WASD` - движение
- `E` - взаимодействие

**Попробуйте:**
- Подойдите к Даше (на диване) и нажмите `E`
- Найдите кошку Рекси и нажмите `E`
- Походите за мебелью - персонаж заходит **за** объекты!

## ❓ Частые вопросы

### Где графика?

Игра готова к профессиональным ассетам! Сейчас используются временные плейсхолдеры (цветные прямоугольники и эмодзи).

Чтобы добавить настоящую графику:
1. Скачайте спрайты с https://itch.io/game-assets/free
2. Положите в папку `public/assets/`
3. Перезапустите игру

Подробности в [ASSETS_GUIDE.md](ASSETS_GUIDE.md)

### Как проверить проект?

```bash
./check-project.sh
```

Этот скрипт покажет статус всех файлов и зависимостей.

## 📞 Нужна помощь?

### Игра не запускается

```bash
rm -rf node_modules package-lock.json
npm install
npm run dev
```

### Порт 3000 занят

Измените в `vite.config.js`:
```javascript
server: {
    port: 3001  // Другой порт
}
```

### Ассеты не загружаются

Проверьте:
1. Файлы находятся в `public/assets/`
2. Имена файлов совпадают с указанными в коде
3. Формат PNG с прозрачностью
